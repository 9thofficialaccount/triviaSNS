# ✅ デプロイメントチェックリスト

公開前に必ず確認してください！

## 🔐 セキュリティ

- [ ] `.env.local` と `.env.production.local` が `.gitignore` に含まれている
- [ ] `NEXTAUTH_SECRET` が32文字以上のランダムな文字列
- [ ] 本番データベースURLに `?sslmode=require` が含まれている
- [ ] デフォルトパスワード（password123）を変更
- [ ] テストアカウントを削除または無効化

## 📦 ビルドとデプロイ

- [ ] `npm install` でエラーが出ない
- [ ] `npm run build` でエラーが出ない
- [ ] `npx prisma generate` が正常に完了
- [ ] データベースマイグレーションが完了（`npx prisma migrate deploy`）
- [ ] 環境変数がVercelに正しく設定されている

## 🗄️ データベース

- [ ] データベースが作成されている
- [ ] データベース接続が確認できる
- [ ] マイグレーションが完了している
- [ ] 初期データ（シード）が投入されている（オプション）
- [ ] バックアップ設定が有効

## 🎨 コンテンツ

- [ ] サンプル投稿が表示される
- [ ] 新規ユーザー登録が動作する
- [ ] ログイン/ログアウトが動作する
- [ ] 投稿作成が動作する
- [ ] 通報機能が動作する
- [ ] コミュニティノート機能が動作する

## 👤 管理者

- [ ] 管理者アカウントが作成されている
- [ ] `/admin` ページにアクセスできる
- [ ] 通報の審査ができる
- [ ] 管理者以外は `/admin` にアクセスできない

## 🔍 検索とナビゲーション

- [ ] タイムラインが表示される
- [ ] 検索機能が動作する
- [ ] トレンドタグが表示される
- [ ] サイドバーが正しく表示される

## 📱 レスポンシブ

- [ ] デスクトップで正常に表示される
- [ ] タブレットで正常に表示される
- [ ] モバイルで正常に表示される

## 🚀 パフォーマンス

- [ ] ページの読み込みが速い（3秒以内）
- [ ] 画像が最適化されている
- [ ] 不要なコンソールログがない
- [ ] エラーが表示されていない

## 📊 モニタリング

- [ ] Vercel Analyticsが有効
- [ ] エラーログが確認できる
- [ ] データベースのパフォーマンスが確認できる

## 🌐 ドメインとSSL

- [ ] HTTPSが有効（Vercelは自動）
- [ ] カスタムドメインが設定されている（オプション）
- [ ] `NEXTAUTH_URL` が正しいドメインを指している

## 📄 ドキュメント

- [ ] README.mdが最新
- [ ] デプロイ手順書が含まれている
- [ ] 利用規約が準備されている（オプション）
- [ ] プライバシーポリシーが準備されている（オプション）

## 🧪 テスト

- [ ] ユーザー登録 → ログイン → 投稿 の流れをテスト
- [ ] 複数のブラウザでテスト（Chrome, Firefox, Safari）
- [ ] 異なるデバイスでテスト
- [ ] エラーハンドリングをテスト

## 📣 公開準備

- [ ] ソーシャルメディアアカウント準備（オプション）
- [ ] プロモーション素材準備（オプション）
- [ ] ユーザーサポート体制（メール、Twitter等）
- [ ] 利用規約とプライバシーポリシー表示

---

## 🎉 すべてチェックが完了したら

**おめでとうございます！公開準備が整いました。**

次のステップ:
1. 本番URLをテスト
2. 友人や知人に共有してフィードバックを得る
3. 定期的にログとアナリティクスを確認
4. ユーザーからのフィードバックに対応

---

## 📞 問題が発生したら

1. Vercelのログを確認: Vercel Dashboard → Deployments → Logs
2. データベースの状態を確認: `npx prisma studio`
3. GitHubでIssueを作成
4. `DEPLOY_GUIDE.md` のトラブルシューティングを参照
