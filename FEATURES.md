# トリビアSNS - 機能一覧

## ✅ 実装済み機能

### 1. コア機能

#### 3層構造の投稿システム
- **第1層（前段）**: 130字以内で知識の出どころを人間味のある形で表現
- **第2層（トリビア）**: 130字以内で核心的なトリビア内容
- **第3層（引用元）**: URL、書籍、論文などの参考文献を記載

#### 投稿作成
- 3層すべてが必須の入力フォーム
- リアルタイム文字数カウンター（130字制限）
- 複数の引用元を追加可能
- 引用元の種類（URL、書籍、論文、記事、その他）を選択

### 2. 信頼性管理

#### 通報システム
- 投稿に対する通報機能
- 通報理由の選択（誤情報、引用元不適切、不適切なコンテンツ、スパム、その他）
- 詳細説明の記入（任意）
- 一定数の通報で自動的に審査中ステータスに移行

#### コミュニティノート
- Twitterスタイルのコミュニティノート機能
- 投稿に対する補足情報や異なる視点を追加
- 追加の引用元を記載可能
- 有用性評価システム（役立つ/役立たない）
- 承認されたノートのみ表示

#### 管理者ダッシュボード
- 通報の確認と審査
- ステータス別フィルタリング（保留中、審査中、解決済み、却下）
- 通報の解決・却下処理
- 審査メモの記録

### 3. タグとカテゴリ

#### AIタグ付け
- OpenAI APIを使用した自動タグ提案
- フォールバック：キーワードベースのタグ抽出
- 投稿作成時に自動でタグを提案
- サポートカテゴリ：科学、歴史、動物、地理、文化、技術、食、健康、自然、宇宙など

#### ユーザータグ付け
- 手動でタグを追加可能
- タグの検索と選択
- トレンドタグの表示

### 4. タイムラインと発見性

#### ハイブリッドタイムライン
- 時系列ベースの投稿表示
- サンプルデータによるプレビュー
- 無限スクロール対応（実装準備済み）

#### 検索機能
- 投稿内容の全文検索
- タグ検索
- ユーザー検索
- タブ別フィルタリング（すべて、投稿、タグ、ユーザー）

#### トレンド機能
- 過去7日間の人気タグを表示
- 投稿数による自動ランキング
- サイドバーにトレンド表示

### 5. ソーシャル機能

#### インタラクション
- いいね機能（リアルタイムカウント）
- コメント機能（UI実装済み）
- ビューカウント

#### 共有機能
- リンクのコピー
- Twitter共有
- Facebook共有
- 画像魚拓化（スクリーンショット保存）

### 6. UI/UX

#### モダンなデザイン
- Tailwind CSSによるレスポンシブデザイン
- Twitter/X風のインターフェース
- グラデーションとシャドウによる視覚的階層
- アイコンライブラリ（Lucide React）

#### モーダルシステム
- 投稿作成モーダル
- 共有モーダル
- 通報モーダル
- コミュニティノートモーダル

### 7. データベース

#### Prisma ORM
- PostgreSQL対応
- リレーショナルなデータモデル
- マイグレーション管理

#### モデル
- User（ユーザー）
- Post（投稿）
- Source（引用元）
- Tag（タグ）
- PostTag（投稿-タグ関連）
- Comment（コメント）
- Like（いいね）
- Report（通報）
- CommunityNote（コミュニティノート）
- Screenshot（スクリーンショット）

### 8. API

#### REST API
- `/api/posts` - 投稿の取得・作成
- `/api/posts/[id]/report` - 通報
- `/api/posts/[id]/community-note` - コミュニティノート
- `/api/posts/[id]/screenshot` - 画像魚拓化
- `/api/search` - 検索
- `/api/tags/trending` - トレンドタグ
- `/api/admin/reports` - 管理者用通報管理
